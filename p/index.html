Website Awards, CSS Awards And A CSS Gallery

The CSSDA provides website awards and CSS awards in a CSS gallery format showcasing the best websites in the world. The CSSDA (CSS Design Awards) accepts website submissions from web designers, website design studios, design agencies and CSS gallery enthusiasts. The CSSDA performs an important role in the web design community by showcasing brilliant website design work. The CSSDA Website Award takes this initiative one step further by also recognizing the web designers that inspire us all.

Submit Your Website To Win A Website Award

To submit your website to our CSS gallery and be nominated for a CSS website award simply visit the Submit Website page and follow the instructions. Every website design you submit must be coded with CSS and validate to W3C standards. All website design work submitted will be judged for creativity, functionality, originality and usability.

Website Awards and CSS Awards Process

Websites submitted to CSSDA will be in the running to win a CSS award and will be placed on the pending list for our judges to view. Successful nominations will be automatically added to the CSS gallery and go in the running for a website award. Only one web design can win a CSS award per day. Winning website designs will be showcased on the home page of CSS Web Design Awards along with the designer of the website. Winners will be notified by email and be showcased in the CSS gallery until the next winner takes their place. Winning entries will then be showcased in a CSS gallery of past winners.
